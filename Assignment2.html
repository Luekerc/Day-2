Git Commands

'git clone' allows you to take a new repository from Git Hub and place it into a local file you can work on


'git status' lets you see if your last command was successful or if your file is up to date


'git add .' adds/stages all of the files in the current directory.


'git commit' stores the staged changes


'git push' moves the changes to your repository on GitHub


'git pull' allows you to pull what you've previously pushed to Github back into your working files as a sort of "reset"
